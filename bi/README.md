# Pasta BI – Indicadores de Desempenho

Esta pasta contém os recursos do módulo **BI (Business Intelligence)** do sistema.

## Conteúdo

- **bi-icon.png** – Ícone do BI exibido no header (ao lado do ícone IA), usado como link para a página de Indicadores de Desempenho.

## Acesso

O ícone BI aparece na barra superior apenas para usuários com permissão **Relatórios avançados**. O link leva para `pages/bi.php`.
